# ☥ Habesha RasTaurant & Cafe - Digital Menu

A modern, golden-themed digital menu system for Ethiopian restaurants with QR code payment integration.

## ✨ Features

- **Interactive Digital Menu** - Beautifully designed menu with Ethiopian cuisine
- **Telebirr Integration** - QR code and merchant number for mobile payments
- **Bank Transfer Details** - CBE account information with one-click copy
- **Fun Quiz Section** - Entertaining Ethiopian food trivia
- **Mobile Responsive** - Works perfectly on smartphones and tablets
- **Golden Neon Design** - Attractive modern Ethiopian aesthetic

## 🚀 Live Demo

Visit: `https://[your-username].github.io/habesha-restaurant-menu/`

## 📱 How to Use

1. **Update Payment Details** - Edit `TELEBIRR_NUMBER` and `BANK_ACCOUNT` variables in the HTML
2. **Customize Menu** - Modify menu items, prices, and descriptions
3. **Deploy** - Push to GitHub Pages for instant hosting
